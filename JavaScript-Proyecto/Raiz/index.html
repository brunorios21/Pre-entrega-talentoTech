Bienvenido

Esta página está pensada para repasar los temas aprendidos en clase sobre JavaScript y los paradigmas de programación. A través de ejemplos y explicaciones sencillas, podrás comprender cómo los objetos, las clases y el polimorfismo le dan vida a nuestros programas.

Clases y Objetos

Una clase es como un molde para crear objetos. Por ejemplo, podemos definir una clase Gato con sus atributos y métodos:


class Gato {
  constructor(nombre, color) {
    this.nombre = nombre;
    this.color = color;
  }

  maullar() {
    console.log(this.nombre + " dice: ¡Miau!");
  }
}

// Crear un objeto a partir de la clase
const michi = new Gato("Michi", "gris");
michi.maullar();

La clase define las características generales y cada objeto creado es una instancia única de esa clase.

Polimorfismo

El polimorfismo significa que un mismo método puede comportarse de diferentes formas según el objeto.


class Animal {
  hacerSonido() {
    console.log("Sonido genérico...");
  }
}

class Perro extends Animal {
  hacerSonido() {
    console.log("¡Guau!");
  }
}

class Gato extends Animal {
  hacerSonido() {
    console.log("¡Miau!");
  }
}

const animales = [new Perro(), new Gato()];
animales.forEach(a => a.hacerSonido());

Con esto, el mismo método hacerSonido() se adapta al tipo de objeto.

Desafío: Juego con Personajes y Elementos

Este desafío propone modelar un pequeño juego en el que personajes y elementos interactúan. Cada objeto tiene propiedades y comportamientos específicos.

  • Luisa: jugadora que maneja personajes.
  • Floki: guerrero con armas como ballesta o jabalina.
  • Mario: trabajador que recolecta y construye.
  • Elementos: castillo, vaca “Aurora” y árbol “Tipa”.

Ejemplo de estructura básica:


class Castillo {
  constructor(defensa) {
    this.defensa = defensa;
  }

  recibirAtaque(potencia) {
    this.defensa -= potencia;
    console.log("Defensa restante:", this.defensa);
  }
}

const castillo = new Castillo(150);
castillo.recibirAtaque(30); // Defensa restante: 120
